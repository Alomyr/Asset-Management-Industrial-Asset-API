# Asset-Management-Industrial-Asset-API
API REST para Gestão de Ativos Industriais desenvolvida com FastAPI e SQLAlchemy. Este projeto foi desenhado para servir de backend em ecossistemas de Automação e Realidade Aumentada (AR), permitindo o controle hierárquico de equipamentos e integração com dispositivos IoT.

for run: python3 -m uvicorn main:app --reload
in browser: http://127.0.0.1:8000/
